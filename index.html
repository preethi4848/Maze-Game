BY🎖️PREETHI G Maze Game LIve Demo🔥 Maze Game project is developed using HTML5, CSS, and JavaScript. Here the player has to escape from maze within given time; that is 30 seconds, movements are done using Keyboard controls ( Arrow Keys or W, A, S, D Keys for movement ). Each level has different mazes which are generated randomly. Now talking about the environment, the game design is set on Canvas. During the gameplay, whenever a player moves from one place to another there are hover colors left as footprints and each movement is counted. The player can move as many as he/she want but within a short period of time.
